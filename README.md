## filetChaotic.py



**Lancer la simulation sur GitHub Pages(https://Morganic07.github.io/filetChaotic.py/)**


## Fonctionnalités 
* **Clic Gauche (Maintenir) :** Attire les points du maillage vers le curseur de la souris
* **Clic Droit :** Tranche les liaisons élastiques à proximité du curseur
* **Barre ESPACE :** Déclenche une onde de choc cinétique explosive à partir du centre de l'écran
* **Éclat Dynamique :** Les liens changent de couleur (bleu --> blanc) et se brisent automatiquement s'ils subissent une tension trop forte
* **Touche ÉCHAP :** Quitte l'application (version Python).
